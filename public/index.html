RECIPES API

Users Routes

#Method URL Body Description
GET /users/usersList none Send a token admin and get a list of al users.
Optional queries:
  • perPage : you can select the amount of obj per one page.
  • page : you can select the num of pages you want.
    		•
    GET /users/myInfo none Send a token and get your information, admin can see all informations.
    POST /users 
    {
    name: String,
    email: String,
    password: String
}
    		Sign up - Add a new user
    POST /users/login 
    {
    email: String,
    password: String
}
    		login, the server will respond with a token
    PUT /users/{idEdit} 
    {
    name: String,
    email: String,
    password: String
}
    		Update your user by token, admin can edit all users.
    DELETE /users/{idDel} none Delete your user by token, admin can delete all users.
